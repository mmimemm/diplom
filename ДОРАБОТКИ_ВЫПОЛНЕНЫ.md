## 🚀 Запуск проекта:

1. **Настройка окружения:**
```bash
cd it-mania
npm install
```

2. **Настройка .env:**
```
MONGO_URI=mongodb+srv://<user>:<password>@cluster0.mongodb.net/itmania
JWT_SECRET=любой_секретный_ключ
PORT=3000
```

3. **Инициализация БД:**
```bash
node server/seed.js
```

4. **Запуск сервера:**
```bash
npm start
```

5. **Доступ:**
- Админ: login=`admin`, password=`admin123`
- Сайт: http://localhost:3000
